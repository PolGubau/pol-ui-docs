## Estilos recomendados [#usage]

Para usar todo el potencial de **pol-ui**, se recomienda trabajar junto a [Tailwind CSS](https://tailwindcss.com/).

A continuación veremos como se puede personificar los components de la manera más sencilla.

### Paso 1. Instalar Tailwind CSS

Si ya tiene instalado Tailwind en tu proyecto, salte directamente al [Paso 2](#step-2-configure-tailwind-css).

Para instalar Tailwind CSS, recomendamos seguir la [documentación oficial](https://tailwindcss.com/docs/installation).

Si tu proyecto usa **React** sin frameworks, te enseñamos como podrías hacerlo.

### Paso 2. Configurar Tailwind CSS

Copie esta configuración en la sección de `content` de su archivo `tailwind.config.js`:

```js {4-6}
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/pol-ui/src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Añadiendo la ruta `./node_modules/pol-ui/src/**/*.{js,ts,jsx,tsx}`, tus estilos de Tailwind CSS afectarán también a los componentes de **pol-ui**.

A continuación podemos configurar nuestro tema en la propiedad `theme`.

### Paso 3. Personalizar tu tema

Dentro de `pol-ui` usamos **6 colores principales**, cada uno cuenta con un máximo de **10 sombras** que ahora adaptará a los estilos de su proyecto usando la propuesta `theme.extend.colors`.

1. **primary**: El color principal de la web.
2. **secondary**: Color secundario, usado para fondos, elementos no importantes, sombras...
3. **success**: Color para algo exitoso.
4. **warning**: Color para el aviso de los usuarios.
5. **error**: Color para elementos que tienen consecuencias en el programa.
6. **info**: Color para dar avisos al usuario que se destaquen.

Aún estos colores, puede extender los colores que usará su proyecto, esto no afectará a `pol-ui` pero no hay problema en ampliar los estilos de su interfaz usando la propiedad `theme.extend.colors`.

De la misma manera, puedes extender `keyframes`, `animaciones`, `sombras`, etc.

Se libre de modificarlas como prefiera.

### Ejemplo de un botón

```jsx {6-7}
import {Button} from 'pol-ui'

function Home() {
  return (
    <div class='flex gap-4 mt-4 items-center'>
      <Button>Botón principal</Button>
      <Button color='secondary'>Botón secundario</Button>
    </div>
  )
}
```

Resultado esperado:

<div class='flex gap-4 mt-4 items-center'>
<Button> Botón principal </Button>

</div>
