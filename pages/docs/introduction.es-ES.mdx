# Introducción
 
 
 

**Pol-ui** es una librería completa de componentes, funciones y UI basada en [React](https://react.dev/).

## Características principales

[Pol-ui](https://pol-ui.com) es una librería completa de componentes, funciones y UI basada en [Tailwind CSS](https://tailwindcss.com) y [React](https://react.dev/). Cuenta con una colección de más de 50 componentes responsivos y móviles, incluyendo navegación, botones, formularios, alertas y mucho más. Está construido con [PostCSS](https://postcss.org/) y [Webpack](https://webpack.js.org/), y puede ser fácilmente personalizado para satisfacer tus necesidades específicas.


### Fácil de usar
Uno de los mayores beneficios de usar Pol-ui es lo fácil que es empezar.
Simplemente instala el paquete, importa el componente que necesitas y listo.

#### Soporte para Next.js
Si estás usando Next.js, puedes seguir la [guía de instalación para Next.js](/docs/getting-started/nextjs) para empezar.

#### Modo oscuro
Toda la UI de Pol-ui está diseñada para funcionar en modo oscuro automáticamente.
Si quieres añadir un interruptor de modo oscuro a tu aplicación, puedes seguir la [guía de modo oscuro](/docs/customize/dark-mode).

#### Personalización
Pol-ui tiene un tema por defecto que puedes personalizar.
Los estilos funcionan con clases de Tailwind CSS, por lo que puedes cambiar el tema por defecto añadiendo clases a tu proyecto.
Si quieres personalizar el tema por defecto, puedes seguir la [guía de personalización](/docs/customize/theme).

#### Contribuir

Si quieres contribuir a Pol-ui, puedes seguir la [guía de contribución](/docs/getting-started/contributing).
