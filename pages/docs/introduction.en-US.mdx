## Key features

[Pol-ui](https://pol-ui.com) is a free and open-source UI component library based on [Tailwind CSS](https://tailwindcss.com) and [React](https://react.dev/). It features a collection of over 50 responsive and mobile-first components, including navigation, buttons, forms, alerts, and much more. It's built with [PostCSS](https://postcss.org/) and [Webpack](https://webpack.js.org/), and can easily be customized to meet your specific needs.


### Easy to use
One of the biggest benefits of using Pol-ui is how easy it is to get started. 
Just install the package, import the component you need, and you're good to go.
